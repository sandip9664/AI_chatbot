### **The Project**
A premium, agentic **AI Chatbot** designed with a minimalist "Arctic Frost" aesthetic[cite: 1]. It utilizes **LangGraph** to maintain sophisticated, stateful conversations with persistent memory, ensuring the AI never loses track of the discussion context[cite: 1].

---

### **Setup Procedure**

1.  **Install Requirements**[cite: 1]:
    ```bash
    pip install streamlit langgraph langchain-openrouter python-dotenv
    ```

2.  **Add API Key**[cite: 1]:
    Create a `.env` file in the root folder
