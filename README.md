Project Description
This is a high-performance AI Chatbot that combines a premium "Arctic Frost" user interface with an advanced LangGraph backend[cite: 1]. Unlike traditional chatbots, this system uses a state-machine architecture to handle complex dialogue and maintain a persistent memory of the conversation[cite: 1]. The UI is built for a professional, minimalist experience, featuring glassmorphism effects, a clean monochromatic theme, and a "Zen" focused layout[cite: 1].

How to Run
Install Dependencies: Open your terminal and run:
pip install streamlit langgraph langchain-openrouter python-dotenv[cite: 1]

Set Up API Key: Create a .env file in your project folder and add your OpenRouter key:
OPEN_ROUTER_API_KEY=your_key_here[cite: 1]

Launch the App: Navigate to your project directory and run:
streamlit run app.py[cite: 1
